# 幼师&教师工具专属小红书标题生成助手
本项目是一款专为幼师、中小学教师打造的小红书标题生成工具，基于 Python + Flask + 智谱 AI 开发。项目旨在解决教育类自媒体内容创作中标题灵感匮乏、风格难以统一、创作效率低下的问题，支持学习用户个人爆款标题风格，一键生成适配教育场景的高点击率标题。

本项目为个人独立开发，已实际应用于日常工作及团队协作。

## 项目背景
在幼师、教师相关自媒体（小红书为主）运营过程中，标题质量直接影响作品点击率。长期创作易出现灵感枯竭、风格不统一、个人爆款风格难以复刻等问题。本项目通过学习用户自建的爆款标题库，结合 AI 大模型实现风格模仿与自动化标题生成，有效提升内容创作效率。

## 核心功能
1. 双身份适配：支持幼师、教师两种身份切换，自动匹配对应场景关键词，避免内容混淆。
2. 快捷工具生成：内置多类高频教学工具，点击即可快速生成对应标题。
3. 个人风格学习：读取本地风格库，让 AI 模仿用户专属爆款标题风格。
4. 风格一键优化：支持增加表情、强化网感、突出痛点、精简吸睛等风格调整。
5. 对话记忆功能：支持多轮修改标题，保留上下文。
6. 本地会话管理：支持新建、切换、删除会话，记录保存在本地。
7. 本地安全部署：所有数据不云端上传，保障隐私安全。

## 技术栈
- 后端：Python + Flask
- AI 模型：智谱 AI GLM-4.5-air
- 前端：HTML + CSS + JavaScript
- 环境配置：python-dotenv
- 网络请求：requests

## 快速开始
### 1. 获取项目代码
将项目下载或克隆至本地。

### 2. 安装依赖
在项目根目录执行以下命令安装依赖包：
```bash
pip install flask requests python-dotenv
```

### 3. 配置 API 密钥
1. 在项目根目录新建 `.env` 文件。
2. 文件内填入以下内容：
```env
ZHIPU_API_KEY=你的智谱AI API Key
```

### 4. 创建个人风格库
1. 在项目根目录新建 `my_style.txt` 文件。
2. 将自己的爆款标题写入该文件，用于 AI 学习风格。

### 5. 启动项目
执行以下命令启动服务：
```bash
python app.py
```

### 6. 访问使用
打开浏览器访问：
```
http://127.0.0.1:5000
```

## 使用说明
1. 切换身份：幼师 / 教师，系统自动适配工具列表与场景关键词。
2. 快速生成：点击工具按钮即可生成对应标题。
3. 风格调整：使用风格标签一键优化标题效果。
4. 会话管理：支持多对话保存，方便管理不同主题标题。
5. 自定义工具：支持输入不在预设列表中的工具名称。
